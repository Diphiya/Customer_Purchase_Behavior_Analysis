## Analysis Workflow
1. **Data Loading**
   - Imported raw data into Python
   - Reviewed schema, data types, and structure

2. **Exploratory Data Analysis (EDA)**
   - Summary statistics and distributions
   - Trend analysis and pattern identification
   - Detection of missing values and anomalies

3. **Data Cleaning & Preparation**
   - Handled missing and inconsistent data
   - Standardized formats (dates, categories, numerical fields)
   - Prepared clean tables for SQL analysis and reporting

4. **SQL-Based Analysis**
   - Loaded cleaned data into relational databases
   - Performed analytical queries including:
     - KPI calculations
     - Grouped and time-based analysis
     - Category and segment comparisons
     - Joins across multiple tables

5. **Data Visualization & Reporting**
   - Built an interactive Power BI dashboard
   - Designed visuals to support business decision-making

## How to Run the Project
1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd <project-folder>
   ```
2. **Run Python analysis**
   - Open the Jupyter Notebook
   - Execute EDA and data cleaning steps

3. **Run SQL queries**
   - Import cleaned data into PostgreSQL / MySQL / SQL Server
   - Execute queries from the /sql folder

4. **View Dashboard**
   - Open the Power BI .pbix file
   - Refresh data sources if required
